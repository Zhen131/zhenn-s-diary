# 如果有一天你想知道我是谁，可以从这里开始

欢迎来到我的小天地。

这是我的个人 Obsidian 知识库，也是我的公开日记仓库。某种程度上，我把自己开源了。

这里放着我的想法、经历、关系、焦虑、学习、金钱、生活、失败、挣扎，以及一些暂时还没有想明白的问题。它不是作品集，不是简历，也不是一个精心包装出来的人设。它更像一份还在持续更新的人生档案。

这不是完整的我。

但这是某种程度上的我。

如果有一天你想知道我是谁，可以从这里开始。

## 这是什么

这个仓库是一座用日记和概念节点搭起来的 Obsidian 知识库。

每一篇日记记录一个具体的时刻：我遇到了什么，我怎么想，我怎么理解关系、学习、情绪、成长、金钱、技术和生活。

每一个概念节点则像一块反复被打磨的石头：有些问题不是一天出现的，也不是一天解决的。它们会在不同年份、不同事件里反复回来，最后慢慢连成一张图。

所以这个仓库既是日记，也是心智地图；既是生活记录，也是一个人持续理解自己的过程。

## 怎么阅读

你可以直接在 GitHub 上阅读这些 Markdown 文件。

但如果想看到它真正的结构，我更推荐用 [Obsidian](https://obsidian.md/) 打开这个仓库。Obsidian 的双链、反向链接和知识图谱，能更好地呈现这些日记之间的关系。

推荐入口：

- `00 工具与指南/📚 日记总目录.md`：从时间顺序浏览
- `2026日记/成长轨迹2023→2024→2025→2026.md`：从成长主线浏览
- 各年份文件夹里的概念节点：从某个反复出现的问题开始浏览
- Obsidian 的关系图谱：从连接本身理解这座知识库

如果你只是在 GitHub 上随便点进来，也没关系。读一篇就读一篇，读不下去也没关系。

## 关于公开

我知道，免费的东西常常不被珍惜。

很多真正有价值的东西，因为免费、开放、安静地放在那里，反而容易被忽略。GitHub 上有很多很好的开源项目和学习资料，但很多人宁愿花钱买一种“我正在学习”的感觉，也不一定真的愿意坐下来学习。

这个仓库也是一样。

它公开，不代表它要求被阅读；它真诚，也不代表别人必须珍惜这份真诚。

我把它放出来，只是因为它对我有意义：它能帮我找到自己的一席之地，也能让别人见证我的成长。也许以后某个认识我的人，想更快地了解我，可以从这里开始。

## 这不是完工的东西

这个仓库会随着我的生活继续更新。

它像一块还没有刻完的碑。

不是因为我已经结束了，而是因为我想在还活着的时候，就诚实地记录：我是谁，我怎么想，我怎么变化，我怎么一点点成为今天的我。
