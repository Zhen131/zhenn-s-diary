# 同居后的幸福与第一块GitHub绿砖

## 今天很重要

今天是 **2026年6月18日**。

他妈累死我了。

但今天也是很重要的一天。不是那种惊天动地的大事，而是很多小东西堆在一起，让我突然感觉：我真的在过一种新的生活。

我已经回到了上海，也开始和对象同居了。

6月15日刚好是我们在一起三周年的纪念日。最近我们吃了很多东西，过得很开心。她总觉得我很可爱，我也在她面前越来越像一个小孩子。

以前我可能会觉得这样有点不好意思，好像自己不够成熟。但现在我更愿意把它理解成一种很强的[[../2023日记/亲密关系]]里的安全感：因为我知道在她面前不用一直端着，不用一直证明自己，所以那个柔软、幼稚、需要被喜欢的部分就自然跑出来了。

我很幸福。

我也很喜欢她。

## 同居之后的幸福感

最近最明显的感受，是一种前所未有的幸福。

这种幸福不是单纯的兴奋，而是一种生活真的落到身边的感觉：她在睡觉，我在旁边忙自己的事；我们一起吃东西，一起待着；她觉得我可爱，我也能感受到自己被接住。

这和我过去对[[../2023日记/爱]]的理解连在一起：真正的爱不是需要我一直表演成一个很厉害、很稳定、很完美的人，而是我可以在关系里变得松弛，甚至变得像小孩子。

可能正因为我感受到了强烈的安全感，所以我不用那么用力地防御自己。

这也是[[温柔比鞭子更有力量]]的另一个版本：当我被温柔地看见、喜欢、接住，我反而更有力气继续往前走。

## 财富自由以后，就像今天这样

刚刚写完项目之后，我们准备下楼逛一逛、散散步、买点东西。

出门前我看见她打扮得很好看。她穿着短裤，还有一件质感很好的上衣。我当时就觉得：她本来就很好看，但今天这样真的特别好看。

下楼之前，我问她：

> 宝贝，你有没有想过，以后我们[[../2025日记/财商|财富自由]]了，会过什么样的生活？

我本来已经预设好了这个对话的走向。我以为她会说我们可以买房车、到处旅游，然后我甚至已经在脑子里想，房车会不会不太方便，是不是还要找个地方定居。

但她回答我：

> 就像今天这样。

那一瞬间我真的觉得好幸福好幸福。

我心里冒出来一个特别强烈的念头：

> 我操，这个女人，我他妈一定要娶她。

这句话让我感觉到一种特别深的知足。原来我想象里的[[../2023日记/爱]]，不是一定要抵达某个很远、很豪华、很宏大的生活，而是今天这样：写完项目，一起下楼，看看她今天很好看，问一个关于未来的问题，然后她说，未来就像现在这样。

后来下楼发现下雨了，我们又回来了。

现在我一边看比赛，她在旁边打游戏。

我还是觉得很幸福。

## 今天忙了一整下午

今天她在睡觉，我一直在敲代码。

说是敲代码，其实很大一部分是叫 AI 帮我敲代码。但这并不代表我什么都没做。真正累的是，我要组织想法、判断方向、检查结果、不断调整，让 AI 助手真的变成我能用的工具。

今天我整理好了 **44 个 SKILL**，也就是 AI 助手用的 Markdown 文档。

这件事本身让我感觉到一种新的[[主体性]]：我不是单纯等工具变好，而是在训练、整理、塑造一套属于自己的工作流。AI 可以替我写很多代码，但决定它要往哪里写、怎么写、写出来是否符合我的目标的人，还是我。

## 第一块GitHub绿砖

今天我还第一次点亮了 GitHub 上的第一个小绿框框，也就是 contribution。

这件事说大也不大，甚至“没啥用”。但它对我来说很有成就感。

因为我以前看那些很牛的程序员，他们 GitHub 上的 contribution 基本都是绿色的，一格一格像瓷砖一样铺满。那种画面会让我感觉：他们真的做了很多东西，他们在持续产出。

而今天，我也有了第一块自己的绿砖。

这其实就是一个很小但很直接的[[反馈系统]]。它告诉我：今天不是空过去的。我确实做了事，我确实留下了痕迹。

这块绿砖不代表我已经很厉害，但它代表我开始了。

## 今天的核心感受

今天最核心的感受，是累，但幸福。

我累，是因为我真的忙活了一整下午；我幸福，是因为我好像同时拥有了两种东西：

- 一边是亲密关系里的安全感；
- 一边是自己做事留下痕迹的成就感。

这两种东西合在一起，让我感觉最近这一段时间特别珍贵。

我不只是被爱着，也在继续变强。

我不只是像小孩子一样被喜欢，也在像一个成年人一样慢慢搭建自己的工具、工作流和未来。

今天很累。

但今天很好。
